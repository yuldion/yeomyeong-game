# 여명 (Dawn) — 게임 프로젝트

1929년 경성을 배경으로 한 대체역사 게임. 15살 소년 한서(韓曙)가 독립운동에 휘말리는 이야기.

## 프로젝트 구조

```
yeomyeong-game/
├─ index.html     실행 껍데기 (거의 안 건드림)
├─ assets.js      base64 이미지 모음 (에셋 바뀔 때만 갱신, 무거움)
├─ game.js        로직·좌표 (★ 배치/수치 조정은 여기만)
└─ assets/        원본 png 전체 백업 (폴더구조 유지, 실행엔 안 쓰임)
   ├─ 건물_40종/       건물 40개
   ├─ 캐릭터_게임용/    순사·약방주인·엄마 등
   ├─ 한서_게임용/      한서 스프라이트
   ├─ 배경_한옥담장/    담장·대문 종류
   ├─ 배경_바닥타일/    바닥 타일
   ├─ 배경_벽돌상점/    상점 배경
   └─ 렌더링_규격.txt   확정 스펙 원본
```

에셋 86개 전체가 백업돼 있어서, 앞으로 어떤 씬을 만들든
레포에서 바로 꺼내 쓸 수 있음. 실행용 assets.js는 씬별 필요분만 담음.

## 작업 흐름 (분리 방식)

- **배치·수치 조정** → `game.js`만 수정하고 커밋 (가벼움)
- **에셋 추가/교체** → `assets/`에 png 넣고 `assets.js` 재생성 (가끔)
- `index.html`은 거의 고정

이렇게 나눈 이유: 예전엔 수정할 때마다 이미지까지 통째로 1MB 넘게 다시 만들었는데,
로직만 분리하니 자주 하는 조정이 훨씬 가벼워짐.

## 실행 방법

세 파일(index.html, assets.js, game.js)을 같은 폴더에 두고 `index.html`을 브라우저로 열기.

또는 GitHub Pages를 켜면 URL로 바로 플레이 가능 (모바일 확인도 가능).

## 확정 렌더링 규격

- 담장 높이 150, 겹침 50px, 원본 229 / 밑단 213
- 대문 위치 50.5%, 원본 305 / 밑단 303
- 한서 캔버스 157×262, 캐릭터 높이 105 (발끝 정렬)
- 지면선(GROUND_LINE) 250

## 씬 구조

- **씬 1 — 한서 집 골목**: 좁고 긴 복도형 (2000×520). 현재 작업 중.
- **씬 2 — 마을 (종로 대로/시장/강변/막다른 골목)**: 오픈 맵. 골목 오른쪽 끝에서 로딩 전환. (예정)

## game.js 주요 조정 지점

- `WALL_RUNS` — 담장 구간. `x`(시작), `to`(끝), `gate`(대문 위치 비율), `gateName`
- `ROOFS` — 담장 뒤 지붕. `x`(위치), `drawH`(크기)
- `MAP_W`, `MAP_H` — 맵 크기
